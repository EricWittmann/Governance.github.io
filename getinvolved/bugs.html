If you discover a bug or would like to request a new feature in any of the Overlord components, feel free to open a ticket within any of the following projects. Overlord uses JBoss JIRA for our tracker.

  • S-RAMP: https://issues.jboss.org/browse/SRAMP
  • Design Time Governance: https://issues.jboss.org/browse/DTGOV
  • Runtime Governance: https://issues.jboss.org/browse/RTGOV
  • API Management: https://issues.jboss.org/browse/APIMAN
  • Overlord: (overlord-commons, website, combined installer, etc.) https://issues.jboss.org/browse/OVERLORD
