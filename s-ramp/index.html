Artifact Repository

Store all your artifacts in this easy to use repository; everything from XML to WSDL to PDF documentation.

Atom API

Easily interact with the repository using the S-RAMP specificaton mandated Atom REST API.

Easy Integration

Take advantage of out of the box integration with Maven or design your own using the included Java client.

Multiple Interfaces

Browse the contents of the repository using either the web based browser or the command line client.
